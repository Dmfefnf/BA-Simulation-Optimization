# BA-Simulation-Optimization

Update requirements.txt at the end of the project.

## Structure

- `Simulation/`: normal simulation, scenario files, calibration script and default results workbook.
- `BO/`: Bayesian optimization simulation, BO runner and `bo_results/`.
- `RL/`: reinforcement learning simulation, agents, experiment runner and `rl_results/`.

Old_files folder for files no longer in use or updated. Used for retracing steps if needed. To be deleted at the end of the project.
